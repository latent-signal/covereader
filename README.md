# Cove Reader

This repository is the public home for **Cove**, a work-in-progress native macOS RSS reader.

No application code lives here. Use this repository to:

- report bugs
- request features
- share feedback and ideas in Discussions

## Before opening an issue

- search existing issues to avoid duplicates
- use the bug report template for reproducible problems
- use the feature request template for ideas and enhancements

## Discussions

If your post is exploratory, open-ended, or not yet actionable, please start a GitHub Discussion instead of an issue.

## Status

Cove is an active work in progress, including planned on-device AI integrations.
